Crafting Mechanics Behind Vorici Socket Coloring (PoE Patch 3.24)

Understanding the math and mechanics behind socket coloring will help you make better crafting decisions.

Socket Color Fundamentals

Chromatic Orbs

Socket Color Influence

Higher attribute requirements make it more likely for sockets to roll the corresponding color. In Patch 3.24, socket color probability is calculated using this formula:

probability(color) = (related_attribute + 20) / (sum_of_all_attributes + 60)

For example, for an item with 100 STR, 50 DEX, and 0 INT requirements:

probability(red) = (100 + 20) / (100 + 50 + 0 + 60) = 120 / 210 = 57.1%
probability(green) = (50 + 20) / (100 + 50 + 0 + 60) = 70 / 210 = 33.3%
probability(blue) = (0 + 20) / (100 + 50 + 0 + 60) = 20 / 210 = 9.5%

This explains why getting "off-color" sockets (colors not matching the main attribute of the item) can be extremely difficult and expensive.

Attribute Requirements

Strength/Dex/Int Requirements

Every equipment piece in Path of Exile has attribute requirements that influence socket coloring:

  • Pure STR items (e.g., Astral Plate): Heavily favor red sockets
  • Pure DEX items (e.g., Zodiac Leather): Heavily favor green sockets
  • Pure INT items (e.g., Vaal Regalia): Heavily favor blue sockets
  • Hybrid items (e.g., STR/INT Armor): Have mixed socket color tendencies

In Patch 3.24, the higher the attribute requirement, the stronger its influence on socket color probabilities, making off-color sockets increasingly difficult to obtain.

Off-Color Sockets

What Are Off-Color Sockets?

Off-color sockets are socket colors that don't correspond to the primary attribute requirement of the item:

  • Blue sockets on STR items
  • Red sockets on INT items
  • Green sockets on INT items
  • Any specific combination that doesn't align with the item's attributes

The higher the primary attribute requirement and the more off-color sockets needed, the exponentially more expensive it becomes to craft with Chromatic Orbs.
